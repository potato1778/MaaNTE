# MaaNTE

😊MAA异环小助手😊
由 [MaaFramework](https://github.com/MaaXYZ/MaaFramework) 强力驱动！

本项目还处于早期开发阶段，欢迎提交PR和Issue
</div>

## ✨ 功能一览

- 🎣 自动钓鱼
- 🥤 自动做咖啡
- 💰 自动提取一咖舍收益

## 💡注意事项
游戏需要运行在1280x720分辨率，窗口化，自动钓鱼模式建议设定30帧
自动做咖啡需要设置鼠标输入方式 Seize ，会出现抢占鼠标的情况
